# ArcModule

## Призначення
Рендерить тимчасові електричні «арки» між двома юнітами, що відображають зв’язок умінь (удари ланцюгом, підтримка тощо).

## Залежності
- `SceneObjectManager` — створює та оновлює об’єкти сцени з типом `arc`.
- `getUnitPositionIfAlive` — колбек для отримання актуальних координат юнітів; при відсутності позиції арка видаляється.

## Життєвий цикл
- `initialize()` — без додаткової логіки.
- `reset()` / `load()` — видаляють усі активні арки.

## Основні методи
- `tick(deltaMs)` — пересуває кінці арок разом із юнітами, зменшує залишок життя та видаляє прострочені або «осиротілі» арки.
- `spawnArcBetweenUnits(type, sourceUnitId, targetUnitId)` — створює арку за конфігом із `arcs-db`, налаштовуючи час життя та fade.
- `clearArcsForUnit(unitId)` — прибирає всі арки, що виходять із/входять у конкретний юніт.
- `clearArcs()` — жорстко очищує всі об’єкти модуля.

## Збереження
Стан не серіалізується — арки короткочасні та відтворюються за подіями бою.
