# AchievementsModule

## Призначення
Відстежує досягнення за проходження складних мап і синхронізує їхні бонуси через `BonusesModule`.

## Залежності
- `achievements-db.ts` для конфігурацій досягнень і бонусів.
- `MapModule` (через `syncFromMapStats`) для оновлення рівнів на основі статистики карт.
- `BonusesModule` для активації бонусів.
- `DataBridge` для публікації списку досягнутих ачівментів.

## Життєвий цикл
- `initialize()` — публікує поточний список досягнутих ачівментів.
- `load()` — відновлює рівні ачівментів зі збереження (якщо є) і синхронізує бонуси.
- `reset()` — скидає рівні ачівментів і бонуси.

## Основні методи
- `syncFromMapStats(stats)` — перераховує рівні ачівментів на основі прогресу карт і оновлює бонуси.
- `getLevel(id)` — повертає поточний рівень конкретного ачівменту.

## Збереження та місток
- Зберігає рівні ачівментів у сейві (за наявності).
- Публікує досягнуті ачівки через ключ `achievements/list`, включно з інформацією про бонуси.
